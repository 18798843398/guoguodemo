##### 完善 README.md(文件夹)（参照： https://github.com/wangfupeng1988/wangEditor ）
- README.md 是开源项目的一张脸，用户的第一印象。必须包含以下内容：
- 产品简介（此处要突出特点，打差异化竞争）
- 产品安装和下载
- 快速使用（详细的使用文档或者二次开发文档，外链即可）
- 交流提问区
- 关于作者（放你的博客链接，和收款二维码）
